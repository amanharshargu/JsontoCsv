    Generalized function to convert JSON to CSV

    EXAMPLE OF A JSON File
      [
          {
            "id": 1,
            "name": "Alice",
            "age": 30,
            "city": "New York"
          },
          {
            "id": 2,
            "name": "Bob",
            "age": 25,
            "city": "Los Angeles"
          },
          {
            "id": 3,
            "name": "Charlie",
            "age": 35,
            "city": "Chicago"
          }
        ]

        EXAMPLE OF AN EQUIVALENT CSV FILE
        id,name,age,city
          1,Alice,30,New York
          2,Bob,25,Los Angeles
          3,Charlie,35,Chicago
